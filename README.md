# DAEHO PARK Portfolio
백엔드 개발자 지망생 박대호입니다. 현재 엔코아 아카데미에서 "한화시스템 BEYOND SW CAMP"라는 부트캠프를 진행 중입니다. Java, Spring을 활용한 백엔드 개발을 중심으로 웹 개발에 필요한 기술을 학습하고, 다수의 프로젝트를 진행해 왔습니다. 최종 프로젝트 진행 중이고, 4월 25일 수료 예정입니다.
